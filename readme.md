# Rosangela's Pizza - Official Website

This is the official website for Rosangela's Pizza, a family-owned Italian-American restaurant in Evergreen Park, IL. This project is a simple, fast, and responsive 4-page static website.

This website was built to provide customers with easy access to the restaurant's menu, hours, location, and story. The primary goal is to drive phone orders and dine-in visits.

### Features
* **Fully Responsive:** Mobile-first design that looks great on all devices.
* **Lightweight & Fast:** Built with clean HTML5 and the lightweight Pico.css framework.
* **Accessible:** Semantic HTML for screen readers.
* **Click-to-Call:** Tappable phone numbers for easy ordering.
* **Embedded Map:** Google Maps for easy directions.
